 3. Рекурсия 
Значения по умолчанию для параметров
При  определении  функции  можно  установить  значения  по  умолчанию  для  каждого  параметра.  Если  при  вызове  функции  соответствующий  аргумент  пропущен,  то  используется  значение  по   умолчанию.
 Как   это  сделать  показано  в  следующем  примере.
Параметру  b  присвоено  значение  по  умолчанию  42.   При  вызове  функции  без  передачи  значения  параметру  b  будет  использовано  значение  по  умолчанию.

 
Во  втором  вызове  функции  не  передан  второй  параметр,  поэтому  вместо  него  используется  значение  по  умолчанию.
